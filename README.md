# 社会科学科普

欢迎来到社会科学科普项目！

## 简介

本项目旨在生产、收集、整理和分享与社会科学相关的科普知识，帮助大家更好地理解社会现象、人类行为以及社会结构的运作方式。

## 内容

我们计划涵盖（但不限于）以下领域：
*   经济学
*   社会学
*   政治学
  

## 如何贡献

我们欢迎任何形式的贡献，包括但不限于：
*   提供新的科普文章、书籍，或主题建议
*   校对和修正现有内容
*   分享相关的学习资源

请通过[提出 Issue](https://github.com/zhuganglie/pss/issues)或[发起 Pull Request](https://github.com/zhuganglie/pss/pulls)的方式参与贡献。
